Descrição detalhada

Classe auxiliar para operações gerais.

Esta classe contém diversas funções auxiliares, incluindo manipulação de datas, cálculo de dias entre datas, verificação de anos bissextos e gerenciamento de memória. Ela também permite pausar a tela e introduzir um atraso durante a execução.
