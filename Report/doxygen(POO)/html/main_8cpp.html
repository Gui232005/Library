int main ( )

Função principal do programa.

Este método inicializa uma instância da classe Biblioteca, carrega os dados necessários a partir de arquivos externos (livros, leitores, empréstimos, reservas, multas) e exibe o menu da biblioteca para interação com o usuário.

O programa realiza as seguintes operações:

  • Carrega os livros a partir de "Livros.txt".
  • Carrega os leitores a partir de "Leitores.txt".
  • Carrega os empréstimos ativos, passados, reservas e multas a partir dos respectivos arquivos.
  • Exibe o menu da biblioteca para o utilizador interagir.

Após as operações, o programa termina liberando os recursos utilizados.

Retorna
0 Se a execução for bem-sucedida.
